# HW2_TP - домашнее задание №2. Представляет собой систему управления рецептами
## Описание проекта
Этот проект реализует систему для работы с рецептами, а также составления списка покупок, необходимых для приготовления того или иного рецепта. В проекте используются 4 основных класса:
1. class Ingredient
этот класс является отдельным ингредиентом. У него есть название, его количество и единица измерения
2. class Recipe
этот класс представляет собой рецепт, в котором есть название блюда, которое этот рецепт "готовит", а также список необходимых игридиентов для приготовления. В зависимости от количества порций рецепт меняет количество необходимых игредиентов
3. class ShoppingList
этот класс представляет собой список покупок (необходимых ингреиентов) для приготовления блюд. Их может быть несколько или одно, но ноль или меньше нуля быть не может
4. class DietaryRecipe
этот класс наследуются от классса Recipe, имеет тип диеты, также хранит в себе название блюда, список ингридиентов.
## Использование. Как запустить тесты
Все тесты для каждого класса находятся в файле test_recipes. Для запуска тестов необходимо вставить в терминал команду "python -m pytest test_recipes.py". Эта команда запустит сразу все 18 тестов, которые есть в проекта
